### Env:
Node v12.22.5
Npm v7.5.2

### Install:
`npm i` from root directory.

### Run test scripts:
#### unit:
`npm run cy:open` for run unit tests.

#### unit nyc:
`npm run cy:nyc` for check coverage.

#### e2e:
- `npm run cy:open` for run tests with browser's options.
- `npm run cy:run` for run tests with headless mode.

#### e2e report:
1. `npm run cy:run`;
2. `npm run cy:allure`.
